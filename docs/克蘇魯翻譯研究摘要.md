# 克蘇魯翻譯研究摘要
> 為 Sunless Sea 漢化翻譯準則設計所做的前期研究

---

## 一、洛夫克拉夫特語言的核心特質

### 1. 刻意選用古舊、冷僻的詞彙
- 原文常用過時字詞（如 "shew" 代替 "showed"、"man of science" 而非 "scientist"）
- 大量使用如 `eldritch`（詭異）、`cyclopean`（巨大的）、`blasphemous`（褻瀆的）這類有分量的形容詞
- 目的是讓讀者「感受到一個從霉味舊房子裡冒出來的陰森勁」——語言本身就是氣氛的載體

### 2. 「不可名狀」是技法，不是退路
- 洛夫克拉夫特刻意讓怪物和恐怖保持模糊，**暗示遠比描寫更有力**
- 最常見的語句是「無以名狀」——但這不是懶惰，是哲學立場：人類語言本來就不足以描述宇宙真相
- 「讓讀者的想像去填空，才能達到最大恐懼」

### 3. 複雜句型＋知識性語氣
- 長句、多子句、偽學術口吻（像在寫調查報告或日記）
- 理性與超自然相撞所產生的張力，是洛式恐怖的核心

---

## 二、Sunless Sea 的文字特色

Sunless Sea 文字風格不完全等同純洛夫克拉夫特，而是**維多利亞哥德 + 克蘇魯 + 黑色幽默**的混合體：

- 敘述極度精煉——每句話都有重量（35萬字但每個文本單位都很短）
- 第二人稱視角（"You..."）製造沉浸感與命運感
- 對比手法：冷靜的語氣描述駭人的事物，反差製造恐懼
- 常以**片段暗示**而非完整敘述

---

## 三、中文翻譯的核心挑戰

**最大難點**：英文語境下冷僻詞自然帶有「陌生感、古舊感」，但中文一個「不可名狀」大家都沒感覺了——因為語言文化環境不同，讀者的陌生感無法直接移植。

具體問題：
- 英文古舊詞有「積塵感」，中文如何重現？
- 文言詞可以增加距離感與莊重性，但用錯會變成搞笑
- 名詞音譯（如克蘇魯、奈亞拉托提普）本身已有怪異感，但動詞／形容詞的選擇更難

---

## 四、好的中文克蘇魯翻譯策略

| 面向 | 策略 |
|------|------|
| **詞彙** | 使用帶有古典漢語色彩的詞（幽冥、蒼茫、晦暗、深淵），避免現代口語 |
| **模糊恐怖** | 不要把怪物描述清楚，用「其形難辨」「非人所能言說」等方式保留留白 |
| **學術語氣** | 保留第一人稱調查者的口吻，語氣近乎學術報告或日記 |
| **節奏** | 短句製造壓迫，長句堆疊細節與恐懼感，二者交替 |
| **宇宙冷漠感** | 語氣要體現「無關你這個生命個體的存在」，主體是宇宙，不是人 |
| **音響感** | 某些克蘇魯語詞的音譯要讓人讀起來不自在、不順口——這是刻意的 |

---

## 五、Sunless Sea 翻譯的特殊考量

因為 Sunless Sea 多了維多利亞和黑色幽默元素：

- **黑色幽默不能翻死**：原文常用輕描淡寫的語氣說可怕的事，中文要維持這種反差
- **第二人稱「你」的一致性**：整個遊戲是「你」在敘述，語氣要保持冷靜、帶距離感
- **精煉優先**：每個文字框都很小，不能冗長，要在有限字數內保留氣氛
